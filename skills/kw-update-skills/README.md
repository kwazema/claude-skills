# kw-update-skills

Update all external third-party skills to their latest versions.

## Install

```bash
npx skills add kwazema/claude-skills --skill kw-update-skills
```

## What it does

- Checks all skills installed via `npx skills add` for available updates
- Updates them to the latest version from their source repositories
- Does not touch locally authored or manually symlinked skills

## Example

> "Update skills" or "Check for skill updates"

Claude will give you the update command to run in your terminal.

## Triggers

`update skills`, `check for skill updates`, `actualizar skills`, `actualiza las skills externas`

See [SKILL.md](./SKILL.md) for the full workflow.
